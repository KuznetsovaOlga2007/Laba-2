==========================
## Общее описание решения:
### circle
```
import math

def area(r):
    return math.pi * r * r

def perimeter(r):
    return 2 * math.pi * r
```
- _этот код считает площадь и периметр круга_

### rectangle
```
def area(a, b):
    return a*b

def perimeter(a, b):
    return 2*(a+b)
```
- _этот код считает площадь и периметр прямоугольника_
### square
```
def area(a):
    return a * a

def perimeter(a):
    return 4 * a
```
- _этот код считает площадь и периметр квадрата_
### triangle
```
def area(a, h):
    return a * h / 2 

def perimeter(a, b, c):
    return a + b + c
```
- _этот код считает площадь и периметр треугольника_
